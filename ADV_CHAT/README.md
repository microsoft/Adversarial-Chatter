# ADV-CHAT

## Description
>
>ADV-CHATTER Is a network of learning agents, that are specialized in specific areas of security.  The three agents are specialized in a particular area of security such as 'Data Exposure recognition', 'Software and website Security Flaws', 'Social Engineering' Between these three agents, they are what is being called 'Adverserial Chatter', a way to generate a threat vector of what an adversary may see against our own systems, websites, social media platforms.

## Adversarial Chatter Agent

**Description:**
Adversarial chatter Agent is the main AI Agent, that can scrape the GlitchyWeb Blog site to be prompted for social engineering weaknesses of the individuals, to better train these individuals against a social engineering attack.

**Note:**
>Adversarial Chatter Agent, can be modified to scrape alternate web sources to perform similar social engineering training benefit.

## Psych Agent

**Description:**
The Psych Agent is an Agent that can be used to ingest survey data and generate a BIG5 metric (**Five-Factor-Model**). Includes the five dimensions _Openness to Experience_, _Conscientiosness_, _Extraversion_, _Agreeableness_, _Nueroticism_. Describes personality structure, predictive behavior, can serve as a foundation for studies in personality psychology, developmental psychology and even neuroscience.
